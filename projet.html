Mes projets

Voici l’un de mes projets réalisés dans le cadre du BTS SIO option SLAM. Il s’agit d’un site web de gestion automobile relié à une base de données MySQL.

Projet : Site web de gestion automobile

Ce projet consistait à développer une application web permettant de gérer un parc automobile : ajout, modification et suppression de véhicules.

  • Langages utilisés : HTML, CSS, PHP, SQL
  • Base de données : MySQL (via phpMyAdmin)
  • Outils : Visual Studio Code, XAMPP, GitHub
  • Réalisation : Création solo
  • Durée : 1 semaine

Ce projet m’a permis de mettre en pratique la modélisation Merise, la structuration d’une base de données relationnelle, ainsi que la gestion de formulaires et de sessions utilisateurs.
